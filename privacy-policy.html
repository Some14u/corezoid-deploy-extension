Corezoid Deploy Shortcut Extension

This privacy policy describes how the Corezoid Deploy Shortcut browser extension handles user information.

Information We Collect

The extension collects and stores only the following information:

  • Domain Settings: The list of Corezoid domains you configure in the extension options (e.g., https://admin.corezoid.com)

How We Use Information

The domain settings are used solely to:

  • Determine which websites the extension should activate on
  • Provide keyboard shortcut functionality (Ctrl+S/Cmd+S) for deploying processes in Corezoid

Information Storage

All information is stored locally in your browser using Chrome's storage API. The extension:

  • Does NOT transmit any data to external servers
  • Does NOT collect personal information
  • Does NOT track user behavior
  • Does NOT share data with third parties

Permissions Explanation

The extension requests the following permissions:

  • Storage: To save your domain configuration settings
  • ActiveTab: To access the current page and provide keyboard shortcuts on Corezoid process editor pages
  • Host Permissions: To work with various Corezoid installations on different domains

Data Security

Your domain settings are stored securely in your browser's local storage and are never transmitted over the internet.

Changes to This Policy

We may update this privacy policy from time to time. Any changes will be reflected in the "Last updated" date above.
